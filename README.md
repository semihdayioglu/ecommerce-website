## 🚀 Kurulum

Projeyi yerel makinenizde çalıştırmak için:

1. Projeyi klonlayın:
```bash
git clone https://github.com/semihdayioglu/ecommerce-website.git
```

2. Proje dizinine gidin:
```bash
cd ecommerce-website
```

3. `index.html` dosyasını bir web tarayıcısında açın.
